`README.拡張子`または `readme.拡張子`という名前のファイルは、常にリポジトリのトップページに表示されます。

GitHub では README ファイルは Markdown 形式（拡張子: `.md`）で書くのが慣例となっています。

## 記入例

```
## 見出し
と書くだけで
### 簡単に
1. 構造化された
1. 文書を
1. 書くことができます
1. :octocat:
```
## GitHub 上での表示

## 見出し
と書くだけで
### 簡単に
1. 構造化された
1. 文書を
1. 書くことができます
1. :octocat:

参考: https://github.com/github/markup
